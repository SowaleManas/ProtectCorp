# SME Technology Risk Assessment

## Overview
This project simulates a consulting-style technology risk assessment for a small-to-medium enterprise (SME) operating in retail, warehouse, and office environments.

The purpose of this assessment is to identify technology-related risks affecting confidentiality, integrity, availability, and operational resilience, and to provide practical recommendations aligned with good governance and control practices.

## Scope
The assessment covers the following areas:

- Endpoint security
- Access management
- Patch and vulnerability management
- Backup and recovery
- Network security
- Incident management
- Asset governance
- Third-party and cloud dependency risk

## Objectives
- Identify key technology risks in an SME environment
- Assess the likelihood and impact of each risk
- Evaluate existing controls
- Recommend risk treatments and control improvements
- Demonstrate consulting-style reporting and risk documentation

## Environment Assessed
This scenario assumes an SME with:

- Office users on Windows laptops/desktops
- Shared operational systems
- Warehouse devices and scanners
- Cloud platforms for email, file storage, and business operations
- Internet-connected network infrastructure
- Limited internal IT team support

## Deliverables
- Risk assessment report
- Findings summary
- Recommendations register

## Key Focus Areas
- Technology General Controls (TGCs)
- Access governance
- Processing integrity
- System availability
- Security monitoring
- Operational risk exposure
